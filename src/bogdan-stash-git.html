Bogdan Stashchuk — Полный Курс Git и GitHub Для Начинающих

Git

Распределённая система контроля версий.

Git устанавливается на компьютер или на сервер.

Основная задача - создавать различные версии проекта.

Git работает локально без доступа к интернету.

GitHub

Сервис хостинга git репозиториев.

Преимущества git

  1. Позволяет сохранять различные версии проектов.
  2. Быстрый переход между версиями проекта.
  3. Улучшает процесс командной работы.
  4. Независимая разработка новых функций в проекте.

Проверка версии

git --version

Проверить установлен ли git, и его версию.
